 # The Emeralds Plus Mod
The Emeralds Plus Mod adds lots of new uses for emeralds.It adds the new Emerald Armor, Emerald Tools and the Emerald Upgrade to get the tools and armor upgrade from diamond armor.The Armor and tools strenght is betwenn diamond and netherite

 ##  Emerald Upgrade

The Emerald Upgrade lets you upgrade diamond armor and tools to Emerald Armor and tools with a Emerald block and an Emerald Upgrade.The Emerald Upgrade can be found in a village in a weaponsmith with 50%.
